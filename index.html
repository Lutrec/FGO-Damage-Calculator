# fgo-calculator

Command Reference

Type commands separated by spaces.
Example: nero a44 am30 150000 lancer

  • Servant: Name (nero) or ID (id1)
  • Cards: a1, b2crit, q3, a44 (NP), ex
  • Enemy: HP (100000) and Class (lancer, beast1)
  • Stats: np5, fou2000, ce1000, paw500
  • Core Buffs: atk30, def-20, arts50, np30
  • Other Buffs: power20, crit50, flat500, se150 (Super Effective)
